Projekt GIT

Projekt GIT je náročný :-D

  • git log: mi ukáže co se stalo před tím
  • přejmenování větve z master na main: git branch -m main
  • git init- založení nového repozitáře
  • git add- přidej a vypíšu ten soubor, který chci přidat
  • git ad . - přidej všechno
  • git commit - "napíšu název toho, co přidávám abych věděla, je to název pro mně"
  • git push: pošlu to do toho git hubu
  • git commit -m "název toho, co chci"
  • příkazový řádek otevírám ctrl a středník

GIT je skvělej nástroj !
